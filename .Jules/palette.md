## 2026-07-05 - Materialize CSS Hidden Elements and Screen Readers\n**Learning:** Materialize CSS heavily relies on display: none or structural obscuring for components like floating action buttons (FABs), which means testing tools like Playwright initially see them as hidden and screen readers might miss them entirely. Standard hover interactions also often fail in automated tests for these elements.\n**Action:** Always add aria-labels explicitly and test them using JavaScript mouse event injection to trigger visibility in automated testing tools. Use state="attached" instead of default visibility checks in Playwright.
